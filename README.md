# Task 2: Predictive Analysis using Machine Learning 
## Objective 
The goal of this task was to build a machine learning model to predict survival outcomes based on 
the Titanic dataset. This demonstrates the ability to implement a full ML pipeline including data 
cleaning, feature engineering, and model evaluation. 
## Methodology - **Data Cleaning:** Handled missing values in the 'Age' column using median imputation. - **Feature Engineering:** Selected key features (Class, Sex, Age, Fare) and performed Label 
Encoding on categorical variables. - **Algorithm:** Utilized the **Random Forest Classifier**, an ensemble method known for high 
accuracy and robustness. - **Evaluation:** Evaluated the model using Accuracy Score, Precision, Recall, and a Confusion 
Matrix. 
## Results - **Model Accuracy:** Approximately 80-82%. - **Key Insight:** Gender and Passenger Class were the most significant predictors of survival. 
## Tech Stack - **Library:** Pandas, Scikit-Learn - **Visualization:** Seaborn, Matplotlib
